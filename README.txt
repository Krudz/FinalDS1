/*************************
pasos para ejecutar el proyecto
/*************************

1- abrir el pgadmin y en la base de datos creada y limpia, sin tablas, correr el archivo que "datosSQL.sql"
2- Organizar la librerias en Intellij con Project Structure 
3- Organizar las rutas del fondo de pantalla que adjunto con la imagen "fondoPDF.png" asi como la ruta que seleccionen para generar el PDF
tanto en la clase "PDFactura.java" y "ControladoraReporte.java"
4- Organizar la clave que tengan en pgadmin que coincida con la clave de la clase "ConnectionFactory.java"
5- Ejecutar, lo ideal seria agregar mas productos y proveedores, y hacer facturas como sea posible asi como encontrar fallas.

nota: cualquier duda quedo atento.
